# 都江堰治水 - MVP HTML 原型

## 文件说明

本目录包含基于 `PRD.md` 与 `HTML_UI_Prompts.md` 生成的可交互 HTML 原型，用于验证核心玩法与 UI 布局。

## 文件列表

| 文件 | 说明 |
|------|------|
| `index.html` | 完整单页应用，包含所有场景、弹窗、游戏逻辑与样式。 |
| `README.md` | 本说明文件。 |

## 运行方式

### 方式一：直接打开
双击 `index.html`，使用主流浏览器（Chrome / Edge / Firefox / Safari）打开即可运行。

### 方式二：本地服务器
```bash
# 使用 Node.js
npx serve .

# 或使用 Python
python -m http.server 8080
```
然后访问 `http://localhost:8080/index.html`。

## 原型范围

- **L1「堵」**：完整可玩，包含编辑、放水、模拟、结算。
- **L2「分」**：占位节点，点击显示「开发中，敬请期待」。
- **页面**：BootScene、TitleScene、LevelMapScene、GameScene，以及 5 个 Overlay 弹窗。
- **机制**：石墙（会倒塌）、竹笼（竖放分水、横放阻挡）、金钱系统、节俭判定、撤销/重置、老河工提示。
- **存档**：使用 `localStorage` 保存关卡进度与设置。

## 操作说明

1. 启动后自动进入标题页。
2. 点击「开始治水」进入关卡选择卷轴。
3. 点击 L1 进入游戏，观看开场叙事后关闭弹窗。
4. 在编辑阶段：
   - 点击底部工具栏选择石墙或竹笼。
   - 在画布上点击或拖拽放置构件。
   - 点击已放置构件旋转 90°。
   - 拖拽已放置构件到垃圾桶删除。
   - 使用「撤销」回退上一步，「重置」清空所有构件。
5. 点击「放水」进入模拟阶段，观察水流与村庄状态。
6. 模拟结束后弹出结算，可选择「重试」或「返回关卡」。
7. L1 失败 1–2 次后会触发老河工提示。

## 占位说明

- 所有美术均由 CSS / Canvas 绘制，未使用外部图片资源。
- 所有音效仅通过 `playSound()` 函数标注触发点，未引入真实音频文件。
- 激励视频广告、第三方登录、付费系统均未接入，仅作原则性避让。

## 约束声明

本原型严格遵循 `PRD.md` 的 MVP 范围，未添加以下未授权内容：
- L3–L8 关卡
- 完整碑廊系统
- 三层提示树、复杂剧情
- 真 3D 运行时渲染
- 多人排行榜、社交裂变
- 资源/跳关付费
